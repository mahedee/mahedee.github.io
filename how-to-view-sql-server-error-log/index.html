    How to view SQL Server error log

    less than 1 minute read

    The SQL Server error log contains user-defined events and certain system events. You can use this error log to troubleshoot problems related to SQL Server.

    Step to view SQL Server error log

    • Object Explorer -> Server -> Management -> SQL Server Logs.

    • Right-click a log -> Click View SQL Server Log.
